# Middleman Template
Middleman application template.


## Structure

* Middleman
* Template Engine
  * Slim
* Front
  * TypeScript with ES2015
  * Sass
* Built tools
  * Gulp
  * Webpack


## Directories

```
*
|- assets
|  |- fonts
|  |- images
|  |- javascripts
|  |- stylesheets
|
|- source
|  |- layouts
```


## Development

Run below commands.

```
$ cp -r Middleman /path/to/project
$ cd /path/to/project
$ yarn run init
$ yarn run dev
```
